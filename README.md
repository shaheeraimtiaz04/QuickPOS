**QuickPOS Landing Page**
**Overview**

QuickPOS Landing Page is a fully implemented, responsive landing page for the QuickPOS system.
It is built using HTML, CSS, JavaScript, and PHP. The page includes all essential sections such as hero, features, pricing, contact form, and a thank-you page.

**Features**

Responsive navigation bar
Hero section with headline and call-to-action
Features section (4 main POS features)
Pricing section (Basic, Pro, Enterprise)
Functional PHP contact form with validation
Redirects to a thank-you page after submission
Fully responsive layout for all devices
Footer with social links

**Installation & Running**

Install PHP on your local machine if not already installed.
Open the project folder in VS Code.
Install the PHP Server extension (or use the built-in PHP server).

**Start the server.**

Example using terminal:
php -S localhost:8000

Open your browser and go to:
http://localhost:8000/index.php
